# TechSummit 2026

A 1-day technical event schedule application built with Node.js and Vanilla JavaScript.

## Features
- Full 1-day schedule (10:00 AM - 5:40 PM)
- 6 Technical Deep Dives
- Real-time Category Search
- Responsive Design

## Local Setup
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `node server.js`
4. Open http://localhost:3000
